A simple custom modification I've made of the Tokyo theme from the oh-my-posh project by JanDeDobbeleer.


How to use the theme:

[Powershell]

oh-my-posh init pwsh --config ~/tokyomodded.omp.json | Invoke-Expression

[Bash]

eval "$(oh-my-posh init bash --config ~/tokyomodded.omp.json)"

[More]

https://ohmyposh.dev/docs/installation/customize
